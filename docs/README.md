# PineLib documentation

Canonical documentation for the PineLib 4.0.0 runtime release line.

## Documents

| File | Purpose |
|---|---|
| `ARCHITECTURE.md` | Runtime package structure, ownership boundary, and interaction with generated modules. |
| `COMPATIBILITY.md` | Supported runtime surface, TradingView parity boundaries, and known non-goals. |
| `DEVELOPMENT.md` | Local checks, test workflow, release gate, and contribution notes. |
| `RELEASE_4_0.md` | 4.0.0 release checklist and acceptance notes. |
| `SECURITY.md` | Runtime safety model and integration guidance. |

## Reading order

Start with `ARCHITECTURE.md`, then `COMPATIBILITY.md`, then `DEVELOPMENT.md`. Use `RELEASE_4_0.md` before tagging and `SECURITY.md` before embedding PineLib in a higher-risk runtime path.
